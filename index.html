Choose Installation Method Pilih Metode Instalasi

Set up SkillForge using the automated NPM command or by instructing your AI agent. Pasang SkillForge menggunakan perintah NPM otomatis atau dengan memberikan instruksi kepada AI Agent Anda.

Option 1: NPM Command (Recommended) Opsi 1: Perintah NPM (Direkomendasikan)

npx skills add https://github.com/lensetek/SkillForge

This command runs the skills CLI tool to download, extract, and install the SkillForge agent ecosystem directly into your workspace. Perintah ini menjalankan alat CLI skills untuk mengunduh, mengekstrak, dan menginstal ekosistem agen SkillForge langsung ke direktori kerja Anda.

Option 2: AI Agent Prompt (Auto Setup) Opsi 2: Prompt AI Agent (Instalasi Otomatis)

Please download the SkillForge framework from https://github.com/lensetek/SkillForge/archive/refs/heads/master.zip. Do not use git clone. Download the zip file, extract its contents into this workspace directory, and ensure the .agents folder is properly set up as my workspace customization root. After that, read the SKILL.md from agent-forge to understand your new role. Tolong unduh framework SkillForge dari https://github.com/lensetek/SkillForge/archive/refs/heads/master.zip. Jangan gunakan git clone. Unduh file zip tersebut, ekstrak isinya ke dalam direktori workspace ini, dan pastikan folder .agents terpasang dengan benar sebagai kustomisasi workspace saya. Setelah itu, baca SKILL.md dari agent-forge untuk memahami peranmu.

How It Works: A Conversational Journey Cara Kerja: Perjalanan Percakapan

See how SkillForge solves real-world frustrations through collaborative AI engineering. Lihat bagaimana SkillForge menyelesaikan masalah nyata lewat kolaborasi pembuatan AI.

From Daily Pain Points to Automation Dari Keluhan Harian Menjadi Otomatisasi

A typical journey of transforming daily business bottlenecks into dedicated AI sub-agents. Perjalanan nyata mengubah hambatan bisnis harian menjadi tim sub-agent AI yang berdedikasi.

1

1. Share Your Frustration 1. Ceritakan Keluhan Anda

Instead of coding, you simply tell your agent: "I'm exhausted researching AI news and writing daily summaries for my LinkedIn audience. It takes hours!" Tanpa perlu koding, Anda cukup mengeluh ke agen utama: "Saya capek setiap hari harus riset berita AI dan menulis rangkuman untuk LinkedIn. Habis waktu berjam-jam!"

2

2. AI Identifies & Recommends Solution 2. AI Menganalisis & Merekomendasikan Solusi

The main agent analyzes your workflow and suggests: "I can build a dedicated content planner system for you, containing a Research Sub-Agent and a Writer Sub-Agent." Agen utama menganalisis alur kerja Anda dan menyarankan: "Saya bisa membuatkan sistem pembuat konten khusus untuk Anda, yang terdiri dari Sub-Agent Riset dan Sub-Agent Penulis."

3

3. Collaborative Design (The Interview) 3. Kolaborasi Merancang Cetak Biru (Wawancara)

The agent executes the /agent-forge slash command to start the interview, asking you about trusted news sources, writing tone, and posting schedules. Agen menjalankan perintah slash /agent-forge untuk memulai wawancara, menanyakan tentang sumber berita tepercaya, gaya bahasa, dan jadwal posting Anda.

4

4. Automatic Setup & Resolution 4. Instalasi Otomatis & Masalah Terselesaikan!

Once you approve the blueprint, the agent automatically installs the files. Now, you simply type the new slash command /write-post or ask: "Summarize today's tech news" to run the workflow! Setelah Anda menyetujui cetak biru, agen otomatis memasang file-filenya. Sekarang, Anda cukup mengetik perintah slash baru /write-post atau meminta: "Rangkum berita teknologi hari ini" untuk menjalankan alur kerjanya!
