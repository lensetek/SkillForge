# SkillForge Standardized Deliverables Directory

This folder is the central storage location for all outputs generated by SkillForge agents.

## Purpose
- **Shared Outputs:** All final documents, reports, processed data files (e.g., CSV, Excel, Word), and code deliverables must be saved here.
- **Agent Resumption:** Incoming agents check this folder to resume work on generated files or references.
- **Workspace Cleanliness:** Keeps the workspace root and the `.agents/` configuration directories clean.
